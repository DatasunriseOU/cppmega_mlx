Before

Generated fused M2RNN MSL computed `tz`, then recomputed the same select(exp(...)) for `tanh_cache` and `h_new`. Exp count was 14.

float tz = select(...);
tanh_cache[...] = tvm_bfloat16(select(...));
float h_new = ... select(...);

After TVM CSE

The generated MSL now binds the pure scalar expression once and reuses it. Exp count is 6.

float cse_v1_2 = select(...);
tanh_cache[...] = tvm_bfloat16(cse_v1_2);
float h_new = ... cse_v1_2;

Fixes Applied

Code and environment

  • cppmega_mlx/nn/_tilelang/m2rnn_path_c.py: Path C production VJP uses hand MSL recurrent backward with packed-head gradient reduction.
  • tilelang/3rdparty/tvm/src/tirx/transform/common_subexpr_elim.cc: CSE now permits pure calls based on SideEffect(expr) <= kPure while still rejecting impure calls and BufferLoad.
  • tilelang/3rdparty/tvm/tests/python/tirx-transform/test_tir_transform_common_subexpr_elim.py: regression coverage for pure call CSE and impure call guard.
  • /Volumes/external/sources/nanochat/.venv: reinstalled TileLang from the local checkout after rebuilding `libtvm_compiler.dylib`.
