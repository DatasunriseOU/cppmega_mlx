What This Report Compares

Full local 1B-class training matrix, not a microkernel-only bench.

Workload

  • Model profile: local_gb10_quarter, the 13-layer hybrid profile used by the M0.4 GB10 work.
  • Dataset: real parquet target shard data/parquet_samples/gb10/clang_semantic_4k_v10/val_00000.parquet.
  • Shape: batch 1, sequence length 2048, measured training steps 20, gradient checkpointing enabled by the matrix command.
  • Optimizer axes: adamw, adam8bit, lion, lion8bit, muon, muon_adamw, and muon_int8. The tables also show the runtime optimizer class, so lion means fp32-moment Lion16 while lion8bit is the quantized optimizer that previously reached the 900 tok/s-class baseline.

Measurements

  • tok/s is the receipt-level mean target-token throughput from scripts/m04_train_step.py.
  • first step s is the first recorded step, used as the compile/warmup cost indicator. steady s is the mean over the remaining steps.
  • peak GiB is MLX/Metal allocator high-water from the per-cell receipt, not live tensor residency.
  • active/cache GiB are the receipt's post-run active and allocator-cache memory; these separate real live tensor growth from cache/compile pressure.
  • Warm / B is Path C warm tok/s / Path B tok/s. Default promotion requires at least 97%.

Route Definitions

The dtype label and path label both matter.

BF16 Path B

Runs --dtype bfloat16 with CPPMEGA_KERNEL_PATH=auto. This is the current non-forced baseline route. It may use existing Path B or reference surfaces according to the normal runtime policy.

BF16 Path C

Runs --dtype bfloat16 with CPPMEGA_KERNEL_PATH=path_c. The default matrix keeps Mamba3 on Path C forward plus Path B backward via --mamba3-bwd path_b; pass --mamba3-bwd path_c only for an explicit full-Path-C experiment.

FP8 Path B

Runs --dtype fp8_path_b and forces CPPMEGA_KERNEL_PATH__SPARSE_MLA=path_b plus CPPMEGA_SPARSE_MLA_FP8_ROUTE=path_b. The DSA Sparse-MLA baseline dispatch is recorded as sparse_mla_fp8_reference_path_b; it is an honest non-Path-C FP8 training baseline, not a Path C fallback.

FP8 Path C

Runs --dtype fp8_path_c, CPPMEGA_SPARSE_MLA_FP8_ROUTE=path_c, and the selected Mamba3 backward policy from the matrix command. Sparse-MLA consumes prepared q_fp8/q_scale/kv_fp8/kv_scale buffers through the TileLang/tvm-ffi route.
